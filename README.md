# BaoYu PMS

A comprehensive Property Management System for boutique hotels and homestays.

## Features

- **Property Management**: Manage multiple properties and rooms.
- **Calendar View**: Visual stock and price management.
- **Order Management**: Handle bookings from various channels.
- **Channel Manager**: Sync inventory with Airbnb, Ctrip, Meituan, etc.
- **Reporting**: Financial and operational reports.

## Tech Stack

- **Frontend**: Vue 3, Vite, Element Plus
- **Backend**: Node.js, Express, SQLite/MySQL
- **Mobile**: Vue 3 (H5)
